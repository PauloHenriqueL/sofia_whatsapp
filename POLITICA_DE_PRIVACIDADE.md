# Política de Privacidade - Sofia (Bot WhatsApp)

**Associação Allos**  
**Clínica-Escola de Psicologia**  
**Belo Horizonte, MG**

---

## 1. Introdução

A Associação Allos ("**Allos**", "**nós**", "**nossa**") é comprometida com a proteção de seus dados pessoais e privacidade. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações quando você interage com **Sofia**, nosso assistente digital via WhatsApp.

Esta política está em conformidade com a **Lei Geral de Proteção de Dados (LGPD)** - Lei nº 13.709/2018 e outras legislações aplicáveis.

---

## 2. Quem Somos

**Organização:** Associação Allos  
**Tipo:** Clínica-Escola de Psicologia (sem fins lucrativos)  
**Localização:** Belo Horizonte, Minas Gerais, Brasil  
**Responsável pelos Dados:** Coordenadora Clínica ou Responsável Legal da Allos

---

## 3. Qual Informação Coletamos

### 3.1 Informações Que Você Fornece Voluntariamente

Quando você interage com Sofia via WhatsApp, podemos coletar:

```
Dados de Identificação:
├─ Nome completo
├─ Data de nascimento
└─ Gênero (opcional)

Dados de Contato:
├─ Número de WhatsApp
├─ Telefone de emergência / contato de apoio
└─ Endereço (rua, bairro, cidade, CEP)

Dados Clínicos / Motivo da Busca:
├─ Motivo de interesse em atendimento
├─ Disponibilidade de horários
├─ Preferência de terapeuta (sexo, abordagem)
└─ Histórico breve de motivo da busca (apenas para qualificação)

Conteúdo da Conversa:
├─ Mensagens de texto que você envia
└─ Tipo de mídia (áudio, imagem, documento - metadados apenas)
```

### 3.2 Informações Coletadas Automaticamente

Quando você usa Sofia, podemos coletar automaticamente:

```
Dados Técnicos:
├─ Timestamp de mensagens
├─ Identificador único do WhatsApp (message ID)
├─ Tipo de mensagem (texto, áudio, imagem, etc)
└─ Metadados técnicos (não conteúdo)

Dados de Uso:
├─ Data e hora de interação
├─ Duração da conversa
└─ Estado da conversa (novo, qualificando, etc)
```

---

## 4. Por Que Coletamos Seus Dados

Coletamos e processamos seus dados para:

```
Propósitos Principais:
├─ Fornecer atendimento inicial e qualificação
├─ Coletar dados necessários para cadastro no sistema Hamilton
├─ Criar sua conta como paciente na Allos
├─ Facilitar match com terapeuta adequado
├─ Melhorar qualidade do atendimento
└─ Contato futuro para agendamento de sessões

Propósitos Secundários:
├─ Comunicação sobre políticas, atualizações de serviço
├─ Resolução de problemas técnicos
└─ Análise de qualidade do bot (sem dados sensíveis)

Obrigações Legais:
├─ Conformidade com legislação de saúde
├─ Conformidade com LGPD
└─ Conformidade com políticas da Meta
```

---

## 5. Como Usamos Seus Dados

### 5.1 Processamento de Dados

Seus dados são utilizados por:

```
Internamente (Allos):
├─ Sofia (bot conversacional - automático)
├─ Thainá (coordenadora clínica - manual)
├─ Terapeutas (após cadastro - para atendimento)
└─ Administração (apenas dados necessários)

Externamente:
└─ Meta (apenas número de WhatsApp, timestamp, tipo de msg)
   └─ Conforme Termos de Serviço do WhatsApp Business
```

### 5.2 O Que NÃO Fazemos

```
❌ Não vendemos seus dados
❌ Não compartilhamos com terceiros (exceto Meta/Hamilton)
❌ Não usamos dados pra publicidade/marketing
❌ Não criamos perfis discriminatórios
❌ Não transferimos dados internacionalmente
❌ Não usamos dados pra outros fins sem consentimento
```

---

## 6. Base Legal para Processamento (LGPD)

Processamos seus dados sob estas bases legais:

```
Consentimento:
└─ Você consente ao interagir com Sofia
   └─ Pode revogar a qualquer momento

Execução de Contrato:
└─ Coleta de dados necessários pra cadastro e atendimento

Obrigação Legal:
└─ Conformidade com regulações de saúde

Legítimo Interesse:
└─ Melhoria de serviço e prevenção de fraude
```

---

## 7. Retenção de Dados

### 7.1 Quanto Tempo Guardamos

```
Dados da Conversa:
└─ 90 dias (para análise de qualidade e resolução de problemas)
   └─ Depois: anonimizados ou deletados

Dados de Cadastro:
└─ Enquanto você for paciente + 5 anos (requisito legal)
   └─ Depois: anonimizados conforme legislação

Dados Técnicos (logs):
└─ 30 dias
   └─ Depois: deletados automaticamente

Metadados (horários, timestamps):
└─ 1 ano
   └─ Para análise de performance
```

### 7.2 Direito ao Esquecimento

Você pode solicitar deletion de dados após encerramento do atendimento, exceto quando houver obrigação legal de reter.

---

## 8. Segurança de Dados

Implementamos medidas de segurança técnicas e administrativas:

```
Proteção Técnica:
├─ Criptografia SSL/TLS em trânsito
├─ Banco de dados criptografado em repouso
├─ Senhas hasheadas (nunca armazenadas em texto)
├─ Validação de assinatura de webhook Meta (HMAC SHA256)
└─ Backup automático e recuperação

Proteção Administrativa:
├─ Acesso restrito a dados sensíveis
├─ Autenticação HTTP Basic no painel
├─ Logs de auditoria
├─ Política de segurança e treinamento
└─ NDAs com todos os envolvidos
```

## 9. Seus Direitos (LGPD)

Conforme a LGPD, você tem direito a:

```
Direito de Acesso:
└─ Saber que dados temos seus

Direito de Retificação:
└─ Corrigir dados incorretos

Direito de Exclusão:
└─ Pedir deleção de dados (quando possível)

Direito de Oposição:
└─ Se opor a processamento (sob certas condições)

Direito à Portabilidade:
└─ Receber cópia de seus dados em formato estruturado

Direito a Não Ser Discriminado:
└─ Não sofrer discriminação por exercer direitos LGPD
```

**Como exercer seus direitos:**  
Envie email para: **[email de contato Allos]** com:
- Seu nome e número de WhatsApp
- Qual direito você quer exercer
- Detalhes da solicitação

Responderemos em até **15 dias úteis**.

## 10. Compartilhamento de Dados

### 10.1 Com Terceiros

```
Hamilton (Sistema Clínico):
└─ Dados de cadastro (nome, telefone, endereço, etc)
   └─ Necessário pra criar conta e atendimento
   └─ Hamilton tem sua própria política de privacidade

Meta/WhatsApp:
└─ Número de WhatsApp, timestamps, tipo de mensagem
   └─ Conforme Termos de Serviço Meta
   └─ Meta processa como Data Processor

Render (Hospedagem):
└─ Dados armazenados em servidor Render
   └─ Render é Data Processor e cumpre GDPR/LGPD
```

### 10.2 Com Terapeutas

Após cadastro, dados básicos são compartilhados com seu terapeuta (necessário para atendimento):
- Nome, telefone, endereço
- Horários disponíveis
- Motivo da busca
- Preferências (não obrigatório)

```

   └─ Dados podem transitar fora do Brasil
   └─ Meta cumpre compromissos de proteção
```
